# File Organizer

Organize files in a folder based on type.

## Usage

Compile the program.

```sh
go build organize.go
```

### Example

**Organize Files in a Folder**

```sh
organize D:/myfolder
```

Output:

```
6 image files moved
2 video files moved
3 document files moved
No audio files moved
1 compressed file moved
1 other file moved
```
